# Islombek Matkarimov - Personal Portfolio Website

This is a premium, highly interactive personal portfolio website for **Islombek Matkarimov**, a Full Stack Developer (Frontend + Backend) specializing in React, Node.js, MongoDB, and custom Telegram bots.

## 🚀 Live Demo Features
- **Background Particle Canvas**: High-performance HTML5 canvas particle system.
- **Telegram Bot Simulator**: Interactive phone mockup demonstrating `@chef_food_xiva_bot` (restaurant menu, shopping cart, and mock order submission).
- **Backend Flow Visualizer**: Live step-by-step pipeline showing how Node.js and MongoDB process incoming Webhook API requests.
- **Developer's Playground**:
  - Live Markdown editor.
  - JSON formatter & minifier with real-time validation checks.
- **Language Switcher**: Seamless toggling between English (EN) and O'zbek (UZ).
- **Interactive Resume Modal**: Slide-up glassmorphic CV formatted for high-quality printing.

## 📁 Project Structure
- `index.html`: Contains semantic page layout, modal markup, and localized texts.
- `styles.css`: Implements CSS grids/flexbox, animations, glowing borders, custom cursors, and responsive media queries.
- `script.js`: Handles animation rendering, chatbot state logic, JSON/Markdown parsing, and navigation events.

## 💻 Technical Details & Stack
- **Frontend**: HTML5, CSS3, JavaScript, ReactJS (concept represented), Bootstrap (styles integrated)
- **Backend & Database**: Node.js, Express, MongoDB (interactive flow diagram)
- **Specialized API**: Telegram Bot API integration
